# Data_Analytics_Dashboard

The **Data Analytics Dashboard** is a full-stack web application designed to analyze, visualize, and manage sales data through an interactive and user-friendly interface. It enables users to monitor key business metrics, explore sales trends, and gain meaningful insights using dynamic charts and dashboards.

The application is built with **Node.js**, **Express.js**, **PostgreSQL**, **HTML**, **CSS**, **JavaScript**, and **Python**, integrating data processing with real-time visualization. The backend provides RESTful APIs for retrieving and managing sales records, while the frontend displays key performance indicators (KPIs), charts, and reports in an organized dashboard. Python is used for additional data analysis and report generation using the Pandas library.

## Features

- Interactive analytics dashboard
- Real-time KPI visualization
- Sales trend analysis
- Category-wise sales distribution
- Search, filtering, and sorting of sales records
- REST API integration
- PostgreSQL database management
- Data analysis using Python (Pandas)
- Responsive user interface using Bootstrap
- Charts and graphs using Chart.js

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Data Analytics:** Python, Pandas, Matplotlib
- **API:** RESTful Web Services

## Dashboard Metrics

The dashboard provides the following business insights:

- Total Sales
- Total Orders
- Average Sales
- Monthly Sales Trends
- Category-wise Sales Analysis
- Recent Sales Records

## Project Objective

The primary objective of this project is to demonstrate how business data can be collected, processed, stored, and visualized through a modern full-stack application. It showcases the integration of database management, backend API development, frontend visualization, and Python-based data analytics to support informed decision-making.

This project serves as a practical example of a business intelligence dashboard and highlights the use of modern web technologies for data-driven applications.
